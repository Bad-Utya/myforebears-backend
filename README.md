# myforebears-backend

 Сайт доступен по ссылке https://rooots.ru/

## Запуск backend'а

1. Скачать репозиторий;
2. В корне проекта создать файл .env, скопировать в него текст из файла .env.example и заменить "change_me" на нужные данные;
3. Запустить Docker;
4. В консоли написать команду ```docker compose up --build```;
5. Backend будет доступен на порту 8001.
